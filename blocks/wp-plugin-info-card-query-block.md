# WP Plugin Info Card Query Block

<figure><img src="../.gitbook/assets/dlxplugins-1674138990172-1x.webp" alt=""><figcaption><p>Two-column Layout for the Query Block</p></figcaption></figure>

When you first insert the block, you will be prompted to enter some query parameters.

<figure><img src="../.gitbook/assets/dlxplugins-1674138946203-1x.webp" alt=""><figcaption><p>Query Based on Several Parameters</p></figcaption></figure>

You can also sort the results by active installs, downloads, or last updated.

<figure><img src="../.gitbook/assets/dlxplugins-1674138970726-1x.webp" alt=""><figcaption><p>Sort the Results Even Further</p></figcaption></figure>

Once you click on `Query and Configure`, you can further customize using the sidebar options.

<figure><img src="../.gitbook/assets/dlxplugins-1674139023979-1x.webp" alt=""><figcaption><p>Set the Sidebar Options for the Query Block</p></figcaption></figure>
