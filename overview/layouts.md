# Layouts

WP Plugin Info Card comes with five different layouts.

### Card Layout

The card layout is a condensed layout that shows some quick stats about the plugin or theme.

<figure><img src="../.gitbook/assets/dlxplugins-1673708087404-1x.webp" alt=""><figcaption><p>Card Layout</p></figcaption></figure>

The Card can be flipped to reveal more information about the plugin or theme.

<figure><img src="../.gitbook/assets/info-card-flip.gif" alt=""><figcaption><p>The Card Layout has a Flip Effect</p></figcaption></figure>

### WordPress Layout

The WordPress layout plays on the default WordPress plugin embed but provides more relevant stats.

<figure><img src="../.gitbook/assets/dlxplugins-1673709057580-1x.webp" alt=""><figcaption><p>WordPress Layout</p></figcaption></figure>

### Large Layout

The Large layout is a larger layout to showcase your plugins or themes in a beautiful bold layout.

<figure><img src="../.gitbook/assets/dlxplugins-1673709188013-1x.webp" alt=""><figcaption><p>Large Layout</p></figcaption></figure>

### Flex Layout

The Flex layout is a layout that stretches and is ideal when you need WP Plugin Info Card to take up the full width of a template.

<figure><img src="../.gitbook/assets/dlxplugins-1673709383034-1x.webp" alt=""><figcaption><p>Flex Layout</p></figcaption></figure>

### Ratings Layout

The Ratings layout puts the plugin rating front-and-center and can quickly show off the quality of a plugin.

<figure><img src="../.gitbook/assets/dlx-plugin-info-card-ratings.jpg" alt=""><figcaption><p>Ratings Layout</p></figcaption></figure>
