## Finding the Admin Settings From the Plugins Screen

If you are on the plugin's screen, search for `WP Piugin Info Card` and click on the `Settings` link.

<figure><img src="../.gitbook/assets/dlxplugins-1673711694754-1x.webp" alt=""><figcaption><p>WP Plugin Info Card on the Plugins Screen</p></figcaption></figure>

## Finding the Admin Settings from the Menu

Alternatively, in the admin menu, head to `Settings->WP Plugin Info Card`.

<figure><img src="../.gitbook/assets/dlxplugins-1673711707449-1x.webp" alt=""><figcaption><p>Settings Menu</p></figcaption></figure>
